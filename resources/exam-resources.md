# Exam Resources

> Curated, public references. None of these are exam braindumps. The goal is to **understand** the material, not memorize questions.

## Official

- [CompTIA Cloud+ (CV0-004) page](https://www.comptia.org/certifications/cloud) — exam objectives, voucher, FAQ.
- [CompTIA CertMaster Practice](https://store.comptia.org/comptia-certmaster-practice/p/APL-PRAC) — official practice questions (paid).
- [CompTIA CertMaster Learn](https://store.comptia.org/comptia-certmaster-learn/p/APL-LEARN) — official eLearning (paid).

## Free High-Quality References

- [AWS Well-Architected Framework](https://aws.amazon.com/architecture/well-architected/) — five pillars, free whitepapers.
- [Azure Architecture Center](https://learn.microsoft.com/en-us/azure/architecture/) — reference architectures and best practices.
- [Google Cloud Architecture Framework](https://cloud.google.com/architecture/framework) — pillars + best practices.
- [Kubernetes Documentation](https://kubernetes.io/docs/home/) — official K8s docs.
- [CNCF Cloud Native Interactive Landscape](https://landscape.cncf.io/) — visualize the ecosystem.
