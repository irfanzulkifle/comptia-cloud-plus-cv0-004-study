# Security Policy

## Supported Versions

This is a documentation / study repository. There is no production code to patch, but the integrity and accuracy of the technical content is taken seriously.

| Branch | Supported |
|---|:---:|
| `main` | ✅ |
| Older branches | ❌ |

## Reporting a Vulnerability or Inaccuracy

If you find:

- A **technical inaccuracy** that could mislead a candidate sitting the exam
- A **security misconfiguration example** that, if copied, would be unsafe
- A **leaked credential, secret, or PII** in any example

Please open a **private security advisory** rather than a public issue:

> GitHub → Security tab → "Report a vulnerability"

We will acknowledge within **72 hours** and publish a fix or correction in the next changelog release.

## Scope

- This repository contains no executable code.
- No PII, real credentials, or production identifiers are (or should be) committed.
- All cloud-provider references are illustrative and intentionally generic.
