# Cloud+ Study Guide

> How to use this repository effectively — both for **exam preparation** and as a **portfolio asset**.

## Audience Modes

| Mode | Read this folder first | Goal |
|---|---|---|
| 🧑‍💼 **Recruiter (60 sec)** | `README.md` | Identify skills, status, professionalism |
| 👨‍💻 **Hiring Manager (5 min)** | `README.md` → `diagrams/` → one objective | Assess depth of cloud knowledge |
| 🧑‍🎓 **Hiring Manager (technical, 15 min)** | `diagrams/` + `objectives/domain-1/1.1-...md` + `cheatsheets/` | Evaluate writing + reasoning |
| 📚 **Study candidate** | `objectives/` → `cheatsheets/` → `flashcards/` → `practice-questions/` | Pass CV0-004 |

## Learning Path

1. **Skim** the [README.md](../README.md) overview.
2. **Read** each objective in `objectives/domain-1/` in order (1.1 → 1.9).
3. **Review** the matching cheatsheet in `cheatsheets/`.
4. **Drill** with `flashcards/` and `practice-questions/`.
5. **Visualize** with `diagrams/`.
6. **Track** your progress in `progress/learning-log.md`.

## Exam-Taking Tips Embedded in the Notes

Every objective file ends with a *Cheat Codes* / *Top Exam Triggers* section — memorize those first.
