## Recruiter-Magnet Tips

1. **First impression = README hero image / first screen.**
   The current README's badges + "Purpose" + "Skills Demonstrated" table hits recruiters in 30 seconds.
2. **Pin it.** Pinned repos are the #1 way a recruiter judges a profile.
3. **Green commits.** A commit graph that grows weekly signals "active learner" more than a single huge push.
4. **One PR from someone else** (or yourself reviewing your own past commit) shows you understand code review.
5. **Domain 1.1 is your showpiece.** It is the largest, deepest note — point recruiters to it.
